# RastimWink
 
Backend API for a team task-management and employee well-being platform, built during a hackathon hosted by **Wink**.
 
## About
 
RastimWink addresses a common problem in fast-moving teams: overworked employees, weak communication between staff and managers, and low motivation to take initiative. The project is a system that helps teams manage tasks, improves the day-to-day relationship between employees and their managers, and adds motivational mechanics that encourage people to work hard, propose ideas, and stay engaged — while also helping surface signs of overwork and burnout before they become a problem.
 
I worked as the **backend developer** on the team, building the API that powers the platform.
 
## Tech stack
 
- **FastAPI** — REST API framework
- **Uvicorn** — ASGI server
- Python
## Running locally
 
```bash
uvicorn app.main:app --reload
```
 
## Status
 
Built during a hackathon under time constraints — functional as a backend proof of concept for the described system.
